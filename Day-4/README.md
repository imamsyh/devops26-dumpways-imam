1. Penjelasan tentang git
- git merupakan salah satu version control system yang mempunyai keunggulan yaitu distributed revision control yang mana penyimpanan tidak hanya berada didalam satu tempat saja, tetapi semua orang yang terlibat akan menyimpan database git
